# Zoo Management System (C++)

This is a modular, object-oriented C++ project that simulates animal management in a virtual zoo. Each animal (e.g., Penguin, Wyrm) is defined as its own class with unique attributes, including place of origin (habitat), which is displayed in the program output.

## Features

- **Object-Oriented Design:**  
  Separate `.cpp` and `.h` files for each animal type and core Zoo logic.
- **Animal Classes:**  
  Includes sample animals such as Penguin and Wyrm, each displaying its habitat in their `displayInfo()` method.
- **Menu System:**  
  Add animals, view all animals, and manage your zoo using a text-based menu.
- **Visual Studio Solution:**  
  Includes `.sln` and `.vcxproj` files for easy building and running in Visual Studio.
